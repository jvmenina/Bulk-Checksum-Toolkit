# Bulk Checksum Toolkit

An OS-agnostic toolkit for directory verification

## record_dir

Records all identifying information about a specfied directory and its contents to a local text file (auto-generated filename).

### Usage

```
record_dir.py [-h] [-v] [-d] [-o [OUTPUT]] path
```

#### Arguments

```
positional arguments:
  path                  path to the target directory

options:
  -h, --help            show this help message and exit
  -v, --verbose         verbose output
  -d, --debug           output all arguments (does nothing)
  -o [OUTPUT], 
    --output [OUTPUT]   path to the directory where the output log will be saved
```

#### Example

- `./record_dir.py -vo "./temp" "./temp/test1"` - Record all info in `./temp/test1` and save output to `./temp` directory.

## compare_dirs

Originally meant to be used for manual file syncing. This script can be used to verify existence (and lack thereof) of files between two similarly structured directories. The `source` argument can be a file generated by `record_dir`.

### Usage

```
compare_dirs.py [-h] [-x] source external_dir
```

#### Arguments

```
positional arguments:
  source             path to the source directory or a file containing checksums
  external_dir       path to the external directory

options:
  -h, --help         show this help message and exit
  -x, --no-checksum  use date modified and size instead of checksums
```

#### Examples

- `python "./compare_dirs.py" "temp/test1" "temp/test2"` - Compare directories `temp/test1` and `temp/test2` using file checksums
- `python "./compare_dirs.py" -x "temp/test1" "temp/test2"` - Compare directories `temp/test1` and `temp/test2` using file modification dates and sizes
- `python "./compare_dirs.py" "temp/test1 - 0000000000.txt" "temp/test3"` - Compare the directories `temp/test2` using all file checksum information detailed in `temp/test1 - 0000000000.txt`
